# Determine Goals from Codomain

Lists all search goals based on codomain tags.

## Usage

``` r
codomain_goals(codomain)
```

## Arguments

- codomain:

  ([paradox::ParamSet](https://paradox.mlr-org.com/reference/ParamSet.html))  
  The codomain to inspect.

## Value

`character`: subset of `"minimize"`, `"maximize"`, `"learn"`. Future
versions may add more tags.
