# Pseudo-Label Active Learning Proposer

Builds a batch sequentially by adding pseudo-labels to a temporary
archive between proposals.

The pseudo-label values are computed by the `label_surrogate_id`
surrogate.

Creates a pseudo-label proposer.

## Arguments

- label_surrogate_id:

  (`character(1)`)  
  Label surrogate registry id.

## Fields

- `label_surrogate_id`:

  (`character(1)`)  
  Label surrogate registry id.
